Content of folders inside "final robot\components\controller":

robot_config:		contains the macros to be initialized before loading the code on the robot
std_chassis_control:	controls the chassis of the standard robot
br_chassis_control:	controls the chassis of the balancing robot
sentry_chassis_control:	controls the chassis of the sentry robot
gimbal_control:		controls the gimbal of all robots
shoot_rev_control:	controls the shooting and rev motors of all robots
control_alg:		contains the implementation of all the control algorithms
control_util:		contains utility functions for control purposes
math_util:		contains utility functions for math/algebric operations
referee_alg:		contains the implementation of all the referee algorithms
temp_pid:		contains the implementation of a PID controller for temperature management (do NOT modify the file)
